# Mastermind

![Mastermind](img/mastermind.png)

## Dependencies

- `CMake` >= 3.12.4
- `openmpi-bin` >= 1.10.2
- `libopenmpi-dev` >= 1.10.2

One of:

- GCC 5
- GCC 6
- GCC 7
- GCC 8
- Clang 6
- Clang 7

## Usage

This project uses CMake and builds "out-of-source":

```
$ mkdir build
$ cd build
$ cmake ..
$ make <some target>
```
